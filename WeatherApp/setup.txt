To set up the enviromental keys for this project create a file called 'app-env' and
include the following:

# run source app-env to load this
export GEOCODE_API_KEY="VALUE"
export WEATHER_API_KEY="VALUE"

Replace the VALUE with the actual keys from the API.

Run the file by using:
 Linux: source app-env
